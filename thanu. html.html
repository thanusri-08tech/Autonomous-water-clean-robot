    About the Project

    The Autonomous Water Surface Cleaning Robot is a software-simulated robotic system designed to autonomously navigate water bodies, detect floating waste, and collect debris using AI-driven navigation. This dashboard demonstrates how telemetry, computer vision and route optimization would integrate in a real deployment — entirely simulated for academic demonstration.
